Initialize Screenbook in a project

## Usage

```
Screen catalog and navigation graph generator (screenbook v0.1.0)
Initialize Screenbook in a project

USAGE:
  screenbook <OPTIONS>

OPTIONS:
  -h, --help                 Display this help message
  -v, --version              Display this version
  -f, --force                Overwrite existing files (default: false)
  --skipDetect               Skip framework auto-detection (default: false)
  --generate                 Auto-generate screen.meta.ts files (--no-generate to skip)
  --no-generate              Negatable of --generate
  --dev                      Start development server after init (--no-dev to skip)
  --no-dev                   Negatable of --dev
  -y, --yes                  Answer yes to all prompts (default: false)
  --ci                       CI mode (no prompts, generate only) (default: false)
  -p, --port [port]          Port for the dev server (default: 4321)

```

## Examples

```bash
screenbook init
screenbook init --force
screenbook init --skip-detect
```
