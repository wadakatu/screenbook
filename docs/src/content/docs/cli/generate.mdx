Auto-generate screen.meta.ts files from route files

## Usage

```
Screen catalog and navigation graph generator (screenbook v0.1.0)
Auto-generate screen.meta.ts files from route files

USAGE:
  screenbook <OPTIONS>

OPTIONS:
  -h, --help                     Display this help message
  -v, --version                  Display this version
  -c, --config <config>          Path to config file
  -n, --dryRun                   Show what would be generated without writing files (default: false)
  -f, --force                    Overwrite existing screen.meta.ts files (default: false)

```

## Examples

```bash
screenbook generate
screenbook generate --dry-run
screenbook generate --force
```
