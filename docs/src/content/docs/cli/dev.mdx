Start the Screenbook development server

## Usage

```
Screen catalog and navigation graph generator (screenbook v0.1.0)
Start the Screenbook development server

USAGE:
  screenbook <OPTIONS>

OPTIONS:
  -h, --help                     Display this help message
  -v, --version                  Display this version
  -c, --config <config>          Path to config file
  -p, --port [port]              Port to run the server on (default: 4321)

```

## Examples

```bash
screenbook dev
screenbook dev --port 3000
```
