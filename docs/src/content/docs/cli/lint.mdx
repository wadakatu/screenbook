Detect routes without screen.meta.ts files

## Usage

```
Screen catalog and navigation graph generator (screenbook v0.1.0)
Detect routes without screen.meta.ts files

USAGE:
  screenbook <OPTIONS>

OPTIONS:
  -h, --help                     Display this help message
  -v, --version                  Display this version
  -c, --config <config>          Path to config file
  --allowCycles                  Suppress circular navigation warnings (default: false)
  -s, --strict                   Fail on disallowed cycles (default: false)

```

## Examples

```bash
screenbook lint
screenbook lint --config screenbook.config.ts
```
